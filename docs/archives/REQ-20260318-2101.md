# REQ-20260318-2101: Font Size Adjustment Feature

## 1. 概要
高解像度（QHD以上）ディスプレイにおいて、アプリケーションのテキストの視認性を向上させるため、フォントサイズをユーザーが調整可能にする。

## 2. 背景
現在のアプリケーションは標準的なフォントサイズ（16px相当）に固定されている。QHD以上のディスプレイではピクセル密度が高いため、相対的にテキストが小さくなり、読みづらいというフィードバックが得られた。

## 3. 機能要件

### 3.1. 設定画面での調整
- 「SYSTEM CONFIGURATION」画面に、フォントサイズを調整する設定項目を追加する。
- 数値入力またはスライダー形式での調整を可能にする（推奨：12px 〜 32px、デフォルト 16px）。

### 3.2. キーボードショートカット
- `Ctrl + +` (または `Ctrl + =`): フォントサイズを拡大する。
- `Ctrl + -`: フォントサイズを縮小する。
- `Ctrl + 0`: フォントサイズを標準（16px）にリセットする。

### 3.3. 設定の永続化
- 変更したフォントサイズは `localStorage` または設定ファイルに保存され、次回のアプリケーション起動時に自動的に適用される。

### 3.4. 全画面への適用
- サイドバー、タスクリスト、エディタ、プレビュー、設定画面など、すべての画面のテキスト、アイコンが適切にスケーリングされること。

## 4. 非機能要件
- パフォーマンス: フォントサイズの変更が即座にUIに反映されること（リロード不要）。
- 互換性: `rem` ベースのスタイリングを主軸とし、絶対値（`px`）指定箇所についても可能な限りスケーリングに対応させる。

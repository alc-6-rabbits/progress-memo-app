# DESIGN-20260318-2101: Font Size Adjustment Implementation

## 1. 概要
`html` 要素、および `body` の `font-size` を CSS 変数を用いて動的に制御する。Tailwind CSS の `rem` 単位によるスケーリング効果を利用し、システム全体を等比級数的に拡大・縮小する。

## 2. 詳細設計

### 2.1. CSS 変数の導入
`assets/css/main.css` の `:root` にベースフォントサイズ変数を定義する。

```css
:root {
  --ace-base-font-size: 16px;
}

html {
  font-size: var(--ace-base-font-size);
}
```

### 2.2. グローバルショートカットと状態管理
`app.vue` または専用の Composables/Plugin にてフォントサイズの状態を管理する。

- **State**: `fontSize` (Number, default 16)
- **Logic**:
  - `increase()`: `fontSize += 2; save();` (Max 32)
  - `decrease()`: `fontSize -= 2; save();` (Min 12)
  - `reset()`: `fontSize = 16; save();`
- **Shortcuts**: 全域（`window.addEventListener('keydown', ...)`）でキャプチャする。

### 2.3. 設定画面の更新
`pages/settings.vue` に「FONT SIZE CONFIGURATION」セクションを追加する。
- 入力UI: 数値入力 + 増減ボタン

### 2.4. スタイル修正
現在、一部の箇所で `text-[10px]` などの絶対値指定が使用されており、これらは `html` の `font-size` 変更に追従しない。これを `rem` ベース（例: `text-xxs` 等を定義）に変更するか、計算式に組み込む。

**案**: `tailwind.config.js` に極小サイズの `rem` 指定を追加する。
- `xxs`: `0.625rem` (10px 相当 / 16px ベース時)
- `tiny`: `0.5rem` (8px 相当)

## 3. 変更ファイル一覧
- `assets/css/main.css`: `:root` 変数と `html` のスタイル定義。
- `tailwind.config.js`: `rem` ベースのフォントサイズ名の拡張。
- `app.vue`: ショートカットリスナーの追加。
- `pages/settings.vue`: UI 項目の追加と保存処理。
- `pages/index.vue` 等: `text-[10px]` などの固定値指定を拡張した `rem` クラスに置換。

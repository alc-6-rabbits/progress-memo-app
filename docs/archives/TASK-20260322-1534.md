## 2. 背景知識: Google Chat のマークダウン仕様

Google Chat は標準 Markdown と異なる独自仕様を持つ。以下の対応表に基づいて変換を実装する。

| 記法 | Google Chat の挙動 | 標準 Markdown との差異 |
|------|-------------------|----------------------|
| `*太字*` | 太字（ボールド） | 標準は `**太字**`（アスタリスク2つ） |
| `_斜体_` | 斜体（イタリック） | 標準と同じ |
| `~取り消し~` | 取り消し線 | 標準は `~~取り消し~~`（チルダ2つ） |
| `` `コード` `` | インラインコード | 標準と同じ |
| ` ```コードブロック``` ` | コードブロック | 標準と同じ |
| `- リスト` | **非対応**（そのまま表示） | 標準はリスト化される |
| `# 見出し` | **非対応**（そのまま表示） | 標準は見出し化される |
| `[text](url)` | **非対応**（そのまま表示） | 標準はリンク化される |
| `> 引用` | **非対応**（そのまま表示） | 標準は引用化される |

## 3. 実装詳細

### 3.1. `composables/useGoogleChatMd.js` 新規作成

```javascript
// composables/useGoogleChatMd.js
export const useGoogleChatMd = () => {

  /**
   * 標準 Markdown → Google Chat Markdown 変換
   * Webhook 送信時に適用する
   */
  const convertToGoogleChat = (standardMd) => {
    if (!standardMd) return ''
    
    return standardMd
      // Bold: **text** → *text*（ただしすでに *text* の場合はそのまま）
      .replace(/\*\*(.+?)\*\*/g, '*$1*')
      // Strikethrough: ~~text~~ → ~text~
      .replace(/~~(.+?)~~/g, '~$1~')
      // リスト記法: - text → ・text（全角ドット）
      .replace(/^- /gm, '・')
      // 見出し記法を除去: # heading → heading（装飾が効かないため）
      .replace(/^#{1,6}\s+/gm, '')
      // 引用記法: > text → ＞ text
      .replace(/^> /gm, '＞ ')
      // リンク記法: [text](url) → text（リンクが効かないため）
      .replace(/\[([^\]]+)\]\([^\)]+\)/g, '$1')
  }

  /**
   * Google Chat プレビュー用 HTML レンダリング
   * Google Chat で装飾が適用された場合の見た目を再現する
   */
  const renderGoogleChatPreview = (text) => {
    if (!text) return ''
    
    // まずGChat形式に変換
    let converted = convertToGoogleChat(text)
    
    // HTML エスケープ
    converted = converted
      .replace(/&/g, '&amp;')
      .replace(/</g, '&lt;')
      .replace(/>/g, '&gt;')
    
    // Google Chat の装飾をHTMLに変換
    converted = converted
      // コードブロック（先に処理）
      .replace(/```([\s\S]*?)```/g, '<pre class="gchat-code-block">$1</pre>')
      // インラインコード
      .replace(/`([^`]+)`/g, '<code class="gchat-inline-code">$1</code>')
      // 太字: *text* → <b>text</b>（ただしリスト記号の * は除外）
      .replace(/(?<!\w)\*([^\*\n]+)\*(?!\w)/g, '<b>$1</b>')
      // 斜体: _text_ → <i>text</i>
      .replace(/(?<!\w)_([^_\n]+)_(?!\w)/g, '<i>$1</i>')
      // 取り消し線: ~text~ → <s>text</s>
      .replace(/(?<!\w)~([^~\n]+)~(?!\w)/g, '<s>$1</s>')
      // 改行
      .replace(/\n/g, '<br>')
    
    return converted
  }

  return { convertToGoogleChat, renderGoogleChatPreview }
}
```

# TASK-20260318-2101: Implement Font Size Adjustment

## 1. 概要
`DESIGN-20260318-2101.md` に基づき、フォントサイズ変更機能を実装してください。

## 2. 実装詳細

### 2.1. ベーススタイルの修正
- `assets/css/main.css` に `:root { --ace-base-font-size: 16px; }` を追加。
- `html { font-size: var(--ace-base-font-size); }` を設定。

### 2.2. Tailwind Config 拡張
- `tailwind.config.js` の `fontSize` プロパティを拡張し、`xxs` (0.625rem), `tiny` (0.5rem) を追加。

### 2.3. グローバルロジックの実装
- `app.vue` または composable にて、`localStorage` から `aceFontSize` を読み込み、`document.documentElement.style.setProperty('--ace-base-font-size', fontSize + 'px')` を実行する。
- 以下のキーイベントを `window` レベルで監視・処理する：
  - `Ctrl + +` / `Ctrl + =`: font-size +2 (Max 32)
  - `Ctrl + -`: font-size -2 (Min 12)
  - `Ctrl + 0`: font-size = 16 (Reset)

### 2.4. 設定画面 (pages/settings.vue) の更新
- 「INTERFACE & STARTUP」セクションに「FONT SIZE (PX)」設定項目を追加。
- 数値入力（12〜32）と「RESET」ボタンを配置。
- 「CONFIRM CONFIGURATION」実行時に即時反映・保存されるようにする。

### 2.5. 既存スタイルの置換
- `index.vue` 等にある `text-[10px]` を `text-xxs` に置換。

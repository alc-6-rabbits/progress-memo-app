# REQ-20260318-2049: Project Overview (Current State)

## 概要
Progress Memo App の現状の機能と技術スタックを定義し、今後の設計・開発のベースラインとする。

## 背景
本プロジェクトは Electron + Nuxt 3 を基盤としたローカル完結型のタスク管理アプリであり、Markdown をデータソースとする。サイバーパンクな UI デザインを特徴とする。

## 技術スタック
- **Runtime**: Electron (v34)
- **Frontend**: Nuxt 3 (Vue 3, Vite)
- **Styling**: Tailwind CSS (Custom "Ace Theme")
- **Storage**: Local Markdown files + JSON (`projects.json`)
- **Integration**: `fs` module via Electron preload/IPC

## 主要機能
1. **DATALINK (Task Management)**
   - Markdown ファイルのリアルタイムパース
   - ステータス別タブ表示（In Progress, Icebox, Archived）
   - 自動ミュート（30日未更新）
   - 外部エディタ同期

2. **Full-screen Editor**
   - リアルタイムプレビュー（2カラム）
   - Markdown ツールバー
   - 自動アコーディオン機能（見出しクリックによる開閉）

3. **Project Management**
   - `projects.json` によるグルーピング
   - タスクへのプロジェクト割り当て

4. **Report Generation**
   - In Progress タスクの集計
   - テキスト出力、Webhook 送信（Google Chat 等）

5. **Storage Settings**
   - フォルダピッカーによる保存先ディレクトリの自由設定

- [ ] 日付入力 UI の不具合修正（過去の経緯より）
